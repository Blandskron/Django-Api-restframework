Bienvenido a la Documentación de la API de Gestión de Personas

Esta API fue creada para gestionar perfiles de personas, permitiendo operaciones CRUD (Crear, Leer, Actualizar, Eliminar) de manera sencilla y eficiente.

Propósito del Proyecto

El propósito de este proyecto es proporcionar una herramienta simple pero poderosa para la gestión de personas. La API puede ser utilizada en diversos contextos, como en aplicaciones de recursos humanos, gestión de usuarios en sistemas de información, y más.

Tecnologías Utilizadas

  • Django: Framework de alto nivel para el desarrollo rápido de aplicaciones web en Python.
  • Django REST Framework: Extensión de Django para crear APIs RESTful de manera sencilla y robusta.
  • Bootstrap: Framework CSS para el diseño responsivo y moderno de la interfaz de usuario.
  • JavaScript: Para la interactividad en el frontend, como la alternancia de la barra lateral y el tema oscuro.

Cómo Empezar

Para comenzar a utilizar la API, navegue a través de las diferentes secciones en la barra lateral. Cada sección proporciona detalles sobre los endpoints disponibles, cómo realizar solicitudes y manejar las respuestas.

Contribuciones

Las contribuciones al proyecto son bienvenidas. Puedes realizar un fork del proyecto en GitHub y enviar tus pull requests para revisión.

Contacto

Si tienes alguna pregunta o comentario, no dudes en contactarnos a través de nuestras redes sociales o correo electrónico.
